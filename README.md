# greensum-client

Client web application for GreenSum Project

## Technologies -

Next  
React  
Semantic UI

## Commands -

1. Install the dependencies in the project folder

    ```shell
    yarn install
    ```

2. Run the development server -

    ```shell
    yarn dev
    ```
